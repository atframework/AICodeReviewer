  - 全流程测试：memoryHintsResolver prompt 注入、postRunCallback 回调、callback 异常安全

## 验收沉淀

- Per-workspace prompt：orchestrator 优先使用 workspace 级 prompt，回退到全局 prompt。
- Force skills：被 `Applies To` 过滤掉的技能在 `force_skills` 配置后重新激活，reason 标记为 `force-activated`。
- Reflection memory：store 表与 API 已就绪，可直接集成到 review lifecycle。
- Memory hints：`memoryHintsResolver` 优先于静态 `memoryHints`；两者都不设时 prompt 中 `{{MEMORY_HINTS}}` 显示 `(none)`。
- Post-run callback：callback 异常不阻塞 review 主流程，仅输出 warn 级别日志。
- Reflection extractor：light mode 从 problems 按 category 归纳模式、从 changed files 归纳文件类型分布、从 summary 归纳标题；thorough mode 最小切片按 `occurrence_count` 识别重复 category；fingerprint 按 workspace+pattern 生成，保证同一 workspace 同一模式的 reflection 可被覆盖更新。
