## 主要交付

### 已交付

- `output_language` 注入 review task context（全局和 per-workspace 可覆盖）
- 所有包 `test/index.test.ts` barrel export 测试（1228 测试全部通过）
- **Per-workspace prompt 覆盖**：`workspaces.instances.<id>.prompt.base_system_prompt_file` 支持指定自定义 system prompt 模板，覆盖全局 `--base-prompt`。Bootstrap 阶段注册 `baseSystemPromptResolver`，orchestrator 在每次 review 前解析。
- **Config-level 强制技能激活**：`workspaces.instances.<id>.prompt.force_skills` 支持按名称强制激活技能，忽略 `Applies To` glob 过滤。Prompt manager 的 `discoverRepoPromptAssets` 接受 `forceSkills` 参数，在 glob 过滤后重新包含被丢弃的强制技能。
- **Reflection memory 存储基础设施**：
  - `reflectionMemory` 表（workspaceId / fingerprint / content / sourceRunId / createdAt / expiresAt）
  - 数据库迁移 `002_reflection_memory`
  - `writeReflectionMemory()`、`readReflectionMemory()`、`compactReflectionMemory()` API
  - 按 workspace 隔离，支持 TTL 过期和条目上限压缩

### 待实现
