## 已固化的关键原则

- 指令前置，system prompt 采用稳定分段结构。
- repo-local 指令由平台装配，不依赖 agent 在源码仓库里自行“碰运气搜索”。
- 高置信问题优先；没有 actionable problem 时优先 `aicr.skip(reason="lgtm")`。
- 输出必须通过 AICR 工具契约流出，而不是自由文本。
- 长篇 repo-local 文档与 skill 正文按需读取，不一次性灌入主 prompt。
- 当 runtime / sandbox 明确内置现代只读工具时，prompt 应优先引导使用
  `rg` / `fd` / `bat` / `jq`，并保留 `grep` / `find` / `cat` 作为兼容兜底。
